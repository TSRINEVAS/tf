# System Details Script

This project contains a small Python script that prints basic system details for the machine where it runs.

## What It Shows

- Current date
- Hostname and IP address
- Operating system details
- Machine and processor information
- CPU count
- Python version
- Current working directory
- Disk total, used, and free space

## Requirements

- Python 3

The script uses only Python standard library modules, so no extra packages are required.

## How To Run

From this project folder, run:

```powershell
python python.py
```

## Example Output

```text
System Details
==============
Date: 2026-06-04
Hostname: COMPUTER-NAME
Ip Address: 192.168.1.100
Operating System: Windows
Os Release: 11
Python Version: 3.14.5
Disk Free: 40.43 GB
```
